This platform toolbox has a number of functions to test and try (these will use your registered JWT for access)
  • Workbench: Log in as any user and browse via UI element or custom list. See API calls being made and stream Box events directly
  • Scopes: Shows how scoped tokens work. Select a user, a file or folder and show differene between scoped and standard token
  • Can view path: See how the 'can view path' collaboration works
  • Webhooks: Show how a live webhook responds. Select user, folder or file and event and webhook payload will be send to the browser
  • Collaboration Report: Generates a (nice) report of all collaborators in a folder structure
The below sandpits use a demo EID for access but you can register and login with any email for testing purposes
  • Okta and Box platform: Use Okta as Identity Management for Box platform solution. Register with Okta and app user is created and mapped.
  • Auth0 and Box platform: Use Auth0 as Identity Management for Box platform solution. Register with Auth0 and app user is created and mapped
  • Cognito and Box platform: Use Cognito as Identity Management for Box platform solution. Register with Cognito and app user is created and mapped
  • AzureB2C and Box platform: Use AzureB2C as Identity Management for Box platform solution. Register with AzureB2C and app user is created and mapped
